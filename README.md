# 🏦 Banking Management System

## Project Overview

The **Banking Management System** is a secure web-based application developed using **Python** and **Django**. It simplifies banking operations by allowing authorized users to manage customer accounts, deposits, withdrawals, fund transfers, and transaction records through an easy-to-use interface.

## Features

* User Authentication (Login & Logout)
* Secure Admin Dashboard
* Customer Management
* Account Creation and Management
* Deposit Money
* Withdraw Money
* Fund Transfer
* Transaction History
* Balance Management
* Input Validation
* Responsive User Interface

## Installation

1. Clone the repository.
2. Open the project folder.
3. Create and activate a virtual environment.
4. Install dependencies:

```bash
pip install -r requirements.txt
```

5. Run database migrations:

```bash
python manage.py migrate
```

6. Start the server:

```bash
python manage.py runserver
```

7. Open your browser and visit:

```text
http://127.0.0.1:8000/
```
